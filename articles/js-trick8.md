## 8. 使用 `Array.prototype.join()`連接數組元素

`Array.prototype.join()`是一種在JavaScript中連接數組元素的方法。以下是一個例子：

```javascript
let words = ['Hello', 'world'];
let sentence = words.join(' ');
console.log(sentence); // 'Hello world'
```

在這個例子中，我們使用 `join()`方法將一個單詞數組連接成一個句子。
